# Chat-Gpt---HSRGUI-MOBILE
이 모드는 지피티를 모바일로 사용할때 테마를 변경하고 싶을때 사용되는 모드입니다. 공식 지피티앱은 지원되지 않아 웹의 확장 프로그램을 이용하여 우회하는 방식으로 사용하는 모드입니다. 따라서 확장프로그램을 지원하는 웹만 가능합니다.

해당 모드의 원본 제작자는 따로 존재합니다.
https://github.com/engineer-502/HSRGUI

모바일 모드는 위 원본 파일을 받아야 실행가능합니다.

만약 문제될시 디스코드로 연락하시면 확인후. 삭제, 비공개 등으로 처리 하겠습니다.
