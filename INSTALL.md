# Installation Guide

## Requirements

You need a browser that supports userscripts.

Recommended:

Android
- Edge Canary

Userscript Manager
- Tampermonkey

---

## Step 1 — Install Tampermonkey

Install Tampermonkey extension.

---

## Step 2 — Install Script

Open Tampermonkey dashboard.

Create a new script and paste the code from:

scripts/HSRGUI_Mobile_0.8.1.user.js

Save the script.

---

## Step 3 — Open ChatGPT

Go to

https://chatgpt.com

Refresh the page.

The HSRGUI mobile interface will appear automatically.

---

## Theme Switching

Use the theme buttons in the header.

EXPRESS  
MIDNIGHT  
SILVER

## Notes

This UI is experimental.

Some browser updates or ChatGPT UI updates may temporarily break compatibility.
