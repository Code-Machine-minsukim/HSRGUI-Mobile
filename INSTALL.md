# Installation Guide

## Requirements

You need a browser that supports userscripts.

Recommended:

Android
- Edge Canary

Userscript Manager
- Tampermonkey

---
## Step 1 — Install Tampermonkey

Install Tampermonkey extension

---

## Step 2 — Install Script

Option1

Open Tampermonkey dashboard.

Go to Utilities Tap and Paste file



Option2

Open Tampermonkey dashboard.

Create a new script and paste the code from:

scripts/HSRGUI_Mobile_0.8.1.user.js

Save the script.

---

## Step 3 — Open ChatGPT

Go to

https://chatgpt.com

Refresh the page.

The HSRGUI mobile interface will appear automatically.

---

## 설치 가이드 (Korean)

## 요구 사항

사용자 스크립트를 지원하는 브라우저가 필요합니다.

권장 환경: Android

앱: Edge Canary

유저스크립트 관리자

Tampermonkey

## 1단계 — Tampermonkey 설치
Tampermonkey 확장 프로그램을 설치합니다.

엣지 카나리에 들어가면 오른쪽 아래 버튼을 누르면 확장프로그램이 보입니다. 들어가서 Tampermonkey 검색후 다운해주세요.

## 2단계 — 스크립트 설치

방법1.

Tampermonkey 대시보드를 엽니다.
새 스크립트를 만든 뒤, 아래 파일의 코드를 붙여넣습니다:
scripts/HSRGUI_Mobile_0.8.1.user.js
스크립트를 저장합니다.

방법2. 추천!

Tampermonkey 대시보드를 엽니다.
도구 탭으로 이동후, 가져오기 클릭하기
HSRGUI_Mobile_0.8.1.user.js
이후 설치 버튼 누르면 됩니다.


## 3단계 — ChatGPT 열기

아래 주소로 이동합니다:
https://chatgpt.com

페이지를 새로고침합니다.

그러면 HSRGUI 모바일 인터페이스가 자동으로 표시됩니다.
